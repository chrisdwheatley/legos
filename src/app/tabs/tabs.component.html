tabs

Our base tabs float with no prominent container around them. There are no borders, just a bottom border on the tab that is selected at the moment.

You can choose where the blue line is, either top or bottom by putting --top or --bottom on each of the labels, ex: <label class="border--top" for="tab1">.

Tab Customizations

One of the customizations for the tabs is having it bordered. If you add --bordered to the tab__container class.

Also this defalut tab component is made for 5 tabs and 5 content areas. If you have more than 5 you are going to need to adjust a couple of things. There is a section in the _tabs.scss file that controls what is shown when you click on a tab.

#tab1:checked ~ #content1,

#tab2:checked ~ #content2,

#tab3:checked ~ #content3,

#tab4:checked ~ #content4,

#tab5:checked ~ #content5 {

display: block;

padding: 20px;

background: $white;

color: #999;

}

You'll just need to add your addtional tabs to your own file. If you have a sixth tab for example you would add : #tab6:checked ~ #content6 and then just extend the %show--tab--content placeholder like below.

#tab6:checked ~ #content6,

#tab7:checked ~ #content7 {

@extend %show--tab--content;

}
