Introduction

It's no secret I'm a LEGO® fan and have been ever since childhood. The name is slightly cheesy and obvious. With reusable pieces of code, I can make components just like using LEGO® bricks to make a castle.

What does this solve:

When I was first learning how to program I used the Twitter Bootstrap and Material Design libraries. They made creating a website painless and easy. On the downside, there was a considerable amount of code that came along that I would never use. I would spend countless hours rewriting specific styles to my liking. When I noticed I was doing this for every project I saw how much time I was wasting and that's how the LegoCSS project was born.

My inspiration:

  • PureCSS.io - the standard for minimal reusable code
  • Github
  • Yelp Styleguide
  • Carbon - IBMs Design Language

... plus many many more.

These guides helped me understand, appreciate, and create this library. I encourage anyone with the same interest to make a library. The decisions and processes are truly valuable to the process of learning.
