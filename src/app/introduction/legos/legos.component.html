legos

The name is somewhat cheesy and obvious. I'm a Legos fan, have been ever since childhood. This library is the coming together of my two passions. Using smallpieces that are reusable and interchangeable to make something larger.

What problem does this project solve

Twitter Bootstrap and Material Design were the style libraries I started coding with when I started coding. They made getting a website going easy and painless. On the downside, there was so much code that came along with those libraries that it would be impossible to really need and use it all on one project. I would spend countless hours overwritting specific styles of theirs to my own preference and doing that for every project. It was quite time consuming, which is how this project came to be.

Some of my inspirations for small modular code have been:

  • PureCSS.io, they have been the standard for minimal reusable code
  • Github
  • Yelp Styleguide
  • Carbon - IBMs Design Language

...plus many many more. They all helped me to understand, appreciate and customize this library to be what I need and maybe what someone else needs. I encourage anyone interested to try to make you're own. The decisions that you need to make and the process you need to go through from start to finish are truely valueable to the process of learning.
