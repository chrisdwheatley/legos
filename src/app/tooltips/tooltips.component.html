tooltip

Our tooltips are pure css and don't rely on on Javascript or jQuery. To add a tooltip follow these instruction:

1. Add a .tooltip__wrapperto the parent of the tooltip.

2. The tooltip itself will be a div or a span with a class of tooltip tooltip--[direction].

Along with the toltip class you need to give the tooltip a direction which will designate where the tooltip will appear from and where the arrow will be located.

Options: tooltip--top, tooltip--right, tooltip--bottom, tooltip--left

Example: Tooltip

<button type="button" class="btn skin-reverse-gray tooltip__wrapper">

Tooltip on left

<div class="tooltip tooltip--left">This is a tooltip on the left</div>

</button>

If you want to change the tooltip color that would be in the tooltipclass.
