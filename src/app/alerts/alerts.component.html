alerts

Alerts are messages that are brought on by different types of user actions. There are two different types of alerts. Use the main alert class on your container.

Attributes: role="alert".

Global Alerts

The global modification of the alert block gets fixed to the top of the screen and stays until dismissed with the close button. This is what differentiates it from a notification alert that fades away after a certain amount of time.

Demo:

* I would have a demo here but it would just stay fixed to the top and that doesn't seem necessary, as it would just sit at the top of this screen.

Markup:
<div class="alert alert--global alert--danger" role="alert">
  ...
</div>

Notifcation Alerts

The notification modification of the alert block fades into an area of the viewport brought on by some type of user interaction. They usually fade-in and out after a certain amount of time. Notifications placement can be customized by adding your own position property to it. The default is: top right.

Demo:
.alert--success
Markup:
<div class="alert alert--notification alert--danger" role="alert">
  ...
</div>

Alert Colors

  • alert--danger
  • alert--success
  • alert--primary
  • alert--warning

These colors control the alerts:

  • Backgroud Color
  • Font Color
  • Close Button Color (only for alert-global)
  • Border Color (optional add-on)
Demo:
alert--danger
alert--success
alert--primary
alert--warning
Markup:
<div class="alert alert--global alert--danger" role="alert">
  <div>Alert text here</div>
  <button type="button" class="alert__close">
    <i class="fas fa-times close" aria-hidden="true"></i>
  </button>
</div>

Border

If you would like to add a border around your alert add the --border modifier to the skin class.

Demo:
alert--danger
alert--success
alert--primary
alert--warning
Markup:
<div class="alert alert--global alert--danger--border" role="alert">
  <div>Alert text here</div>
  <button type="button" class="alert__close">
    <i class="fas fa-times close" aria-hidden="true"></i>
  </button>
</div>
