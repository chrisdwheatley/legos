Striped Tables

To add a zerbra like pattern to the rows add the table--stripe class to the <table> element.

Demo:
# Header Column 1 Header Column 2 Header Column 3
1 Body Column Body Column Body Column
2 Body Column Body Column Body Column
3 Body Column Body Column Body Column
4 Body Column Body Column Body Column
Markup:
<table class="table--stripe">...</table>
