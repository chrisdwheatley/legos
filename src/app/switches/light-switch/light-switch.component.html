Basic

By default the light switch component is a block element. The label is encosed within it so that will always be on the same line as its corresponding switch. The default switch is very much light a light switch, dark and gray when off and golden when on. Colors are customizeable along with the default skins that come with the element.

Demo:
Light Switch
Off On
Markup:
<div class="switch">
  <input type="checkbox">
  <div class="switch__title">label text here</div>
  <span>
    <span>Off</span>
    <span>On</span>
    <a></a>
  </span>
</div>

Inline

Color Custumizations

If you want to customize your switch with a specific color that isn't part of our default skins I recomment making your own skin class, add your styles, to that, and add it alongside the switch element class.

<div class="switch custom-skin">

Inside your stylesheet target these elements:

  • .switch input:checked + span : targets the background color of the switch when its on
  • switch input:checked + span a : targets the color of the circle when switch is on
