Basic

The card component is made with content in mind. It had to be very veritile to take all different types of content in as many different ways that the user throws at it. With that in mind the first demo is classic representation of a card. Image/display area, card title, and text.

Demo:
300 x 168
Card Title This is content for the card. Summary for the card goes here, two to three lines maximum.
Markup:
<div class="card">
  <div class="card-body">
    <span class="card-title">card title</span>
    <span class="card-text">card text</span>
  </div>
</div>

Next we'll walk through what options we can add on and configure.

Images

You can add an image to the top or the bottom of you card by adding a image or div element with a class of.card-display--top or .card-display--bottom and making sure to put it above or below the .card-body section.

Sizes

There are three different sizes that come with the card, unless you'd like to customize the sizes on your own. The default size of the card, if you don't add a size class to it is defaulted to the medium card size. To add a card size to a card add .card--sm or .card--lg.

Alignment

Basic alignment is built in. If you want to do anything further then you'll need to add more custom styles for individual items. Add .card--left, .card--centered, or .card--right on the card element.
