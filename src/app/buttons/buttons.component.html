buttons

Feel free to use an <a>, <span>, <button>, or <input> tag to represent a button. *Thought you should be using best practices and use what is most symantic. As long as you put the .btn class on an <a>, <span>, <input> tag the same styles for a button will be applied to them.

Example: button that looks like a link

<button class="btn link-green">link green</button>

There is also a link-white for dark backgrounds.
