buttons

When making a button please be sure to use a button or a tag for accessability purposes. There are three different type of buttons.

Filled Buttons

Filled buttons have colored borders and backgrounds with a white font color.

Demo:
Markup:
<button class="btn btn--default">...</button>
Outlined Buttons

Outlined buttons only have colored text and borders, otherwise they are the exact same as filled buttons. Same rules apply.

Demo:
Markup:
<button class="btn btn--outline-default">...</button>
Link

While a link does look different than a button, its base is still the same. The text is the color and that's it. No border or background color.

Demo:
Markup:
<button class="btn link--default">...</button>

Block Buttons

For block buttons that span the entire width of the parent container use .btn--block.

Demo:
Markup:
<button class="btn btn--danger btn--block">Default Size</button>
