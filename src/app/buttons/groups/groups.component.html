buttons groups

We also have button groups used for action bar items. Wrap a collection of <button></button> in a <div class="btn--group"> element. Add a skin for colors.



Example: Button Group

<div class="btn--group" role="group" aria-label="...">

  <button type="button" class="btn skin-reverse-gray">Left</button>

  <button type="button" class="btn skin-reverse-gray">Middle</button>

  <button type="button" class="btn skin-reverse-gray">Right</button>

</divp>
