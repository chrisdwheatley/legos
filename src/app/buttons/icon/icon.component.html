buttons with icons

Incorportate Font Awesome icons into your buttons or use an icon as a button.

If you have a combination button that has an icon and a word use the icon class ex: .icon to add some space between your icon and words.

<button class="btn skin-blue">

  <i class="icon fas fa-cog"></i>

  Settings

</button>
upload button

An upload file button is special. We use an input and a label to make a button. You can also add an icon inside.

The .btn--upload class comes along with the button styles that the label needs to look like a button as well as accepting style classes ontop of it like .skin-blue.

Make sure to add the skin color class and or the size class in the correct place. The .skin-{{color}} and .skin-{{size}}go on the label tag. See example below.

<div class="btn-upload-container">

  <input type="file" name="uploadFile" id="upload" class="inputfile">

  <label for="upload" class="btn btn--upload skin-blue skin-md">

    <i class="icon fas icon fa-upload"></i>

    Choose a file

  <label>

</div>
