Chips

Basic

Chips are pill shaped elements. They can be used as list items, labels, or as counters, just as an example. They can be static elements or you can allow users to interact and remove them.

To create chips you need to house them in a .chips container that keeps them all together. Each individual chip gets a .chip class.

Demo:
Hockey Lacrosse Baseball
Markup:
<div class="chips">
  <span class="chip chip-default">Hockey</span>
</div>
Skins

On top of the basic chip you can add the default skin colors to chips. They will alter the background-color, text color, border, and if the chips are removable then the delete icon color as well. To add a skin to the chip just add it to the .chip element. There are two types of skins that can be applied to chips. Just like with buttons, there is a filled chip and and outlined chip. See below for examples.

Demo:
Hockey Lacrosse Baseball Football Soccer
Hockey Lacrosse Baseball Football Soccer
Markup:
<div class"chips">
  <span class="chip skin-green">Hockey</span>
  <span class="chip skin-reverse-blue">Lacrosse</span>
<span class="chip skin-red">Baseball</span>
<div/>
Removable Add-On

It is possible that you want the user to use the chips for a list and be able to remove items as they choose. To do that add the .removable class to the .chip element and the icon element after your chip text. This will add the "x" icon to the end of the chip and allow the user to delete the chip.

Demo:
Hockey Lacrosse Baseball
Markup:
<div class="chips">
  <span class="chip chip-default removable">
    Hockey
    <i class="icon fas fa-times close-chip"></i>
  </span>
</div>
