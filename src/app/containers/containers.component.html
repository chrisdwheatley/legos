containers

There are three classes that can quickly address the layout of content. These are only intended to be used for simple base layouts. For more complex implementation you may need to add more specific css of your own.

Base Container

Just a simplified outer container. The .container class creates a responsive box essentially and adds space on the bottom of each containers.

* Note: If you nest a .container inside of another .container a padding-left: 25px will be added to the inside container. This is to be used as indenting subsections of text or sectioning out different parts of a form, as an example.

Markup:
position: relative
margin: 0 auto 20px auto
padding: 0
width: 100%
Flex Row

Use the .flex-container class as it's own flex container (default row), alongside the base container class to turn it into a flex container, or inside of a base container. This flex container uses the styles below.

Markup:
display: flex
flex-wrap: wrap
justify-content: flex-start
align-items: center
Flex Column

Use the .flex-column container class as it's own flex container with a column format. It does not share styles with the .flex-container class. Like the .flex-container this can be used with or without the base .container class. This flex container uses the styles below.

Markup:
display: flex
flex-direction: column
