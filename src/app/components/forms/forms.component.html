forms

A lot of what we do has to do with styling forms and how all their components fit together. Hopefully these classes can help simplify the process.

Attributes: for="", id="", type="", placeholder=""

Accesability

Forms are made up of inputs, labels, and buttons. Therefore best practices for inputs is the same for forms. Make sure to match labels with their coresponding input using for="" and id="" attributes. Use the placeholder="" attribute to give your users a hint as to how you want them to fill in certain inputs.

Associated classes: form__group. This class groups together label and input elements so that they don't stray from eachother. This is not needed for small inline forms, more so for stacked and aligned forms.

Labels

It is recommend that you use <label> elements in a form and not <div> or <p> tags. The .form__label styles will override even the <p> tag styles but it is better to write semantic code from the beginning.

Default

Our default forms make labels and inputs inline with eachother. By wrapping any form elements such as, inputs and labels, in a form element with the classname .form they will take on the correct styles.

Demo:
Markup:
<form class="form">
  <label for="name">Name:</label>
  <input id="name" type="text" placeholder="...">
  <label for="email">Email:</label>
  <input id="email" type="email" placeholder="...">
  <button class="btn btn--primary" type="button">Submit</button>
</form>

Stacked

To create a stacked form with input elements below their corresponding labels, use form--stacked class on the <form> element

Demo:
Markup:
<form class="form form--stacked">
  <div class="form__group">
    <label for="name">Name:</label>
    <input id="name" type="text" placeholder="...">
  </div>
  ...
</form>

Aligned

For aligned forms there are three different alignments you can choose from (left, middle, and right). To create an aligned form, add the form--aligned-[version] class to the <form> element. On smaller screens the aligned forms will revert back to a stacked form for a better user experience.

Markup:
<form class="form form--aligned-[center]">
  <div class="form form__group">
    <label for="name">Name:</label>
    <input id="name" type="text" placeholder="...">
  </div>
  ...
</form>

Legends

Use a <fieldset> element to visually group together related form elements. An optional <legend> can be used to give the group a name.

Demo:
User
Markup:
<form class="form form--stacked">
  <fieldset>
    <legend>User</legend>
      <div class="form__group">
        <label>Username</label>
        <input type="text" placeholder="...">
      </div>
      <div class="form__group">
        <label>Password</label>
        <input type="password" placeholder="...">
      </div>
      <button class="btn btn--primary" type="button">Submit</button>
  </fieldset>
</form>
