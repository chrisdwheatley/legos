Form

We spend a lot of our online time filling out forms. Whether we are creating a contact form, a shopping cart, or a survey form elements and how they all fit together cohesively are critical to web design. Hopefully these classes can help simplify the process.

Attributes*:
  • for : label element
  • id : input element
  • type : input element
  • placeholder : input element
  • aria-label : input element, defines a label for the element (only used when label text not visible on screen)
  • type : button element
  • title : button element, when the button is just an icon and there is no inner content

* These are used on form element items, not the form element itself.

Accessibility

Forms are made of inputs, labels, and buttons. Make sure to match labels with their corresponding input using for="" and id="" attributes. Use the placeholder="" attribute to give your users a hint as to how you want them to fill in certain input elements.

Associated classes: .form__group.

This class groups together <label> and <input> elements so that they don't stray from each other. This is not needed for small inline forms, more so for stacked and aligned forms.

Labels

It is recommend that you use <label> elements in a form and not <div> or <p> elements. The .form__label styles will override even the <p> tag styles but it is better to write semantic code from the beginning.

Default

Our default forms make labels and inputs inline with eachother. By wrapping any form elements such as, inputs and labels, in a form element with the classname .form they will take on the correct styles.

Demo:
Markup:
<form class="form">
  <label for="name">Name:</label>
  <input id="name" type="text" placeholder="...">
  <label for="email">Email:</label>
  <input id="email" type="email" placeholder="...">
  <button class="btn btn--primary" type="button">Submit</button>
</form>

Stacked

To create a stacked form with input elements below their corresponding labels, use .form--stacked class on the <form> element.

Demo:
Markup:
<form class="form form--stacked">
  <div class="form__group">
    <label for="name">Name:</label>
    <input id="name" type="text" placeholder="...">
  </div>
  ...
</form>

Aligned

There are three alignments you can choose for a form (left, middle, and right). To create an aligned form, add the .form--aligned-[direction] class to the <form> element. On smaller screens, the aligned forms will revert to a stacked form for better user experience.

Demo: Aligned-center
Demo: Aligned-right
Markup:
<form class="form form--aligned-[center]">
  <div class="form form__group">
    <label for="name">Name:</label>
    <input id="name" type="text" placeholder="...">
  </div>
  ...
</form>

Legends

Use the <fieldset> element to visually group related form elements. An optional <legend> ccan be used to give the group a name and bounding box.

Demo:
User
Markup:
<form class="form form--stacked">
  <fieldset>
    <legend>User</legend>
      <div class="form__group">
        <label>Username</label>
        <input type="text" placeholder="...">
      </div>
      <div class="form__group">
        <label>Password</label>
        <input type="password" placeholder="...">
      </div>
      <button class="btn btn--primary" type="submit">Submit</button>
  </fieldset>
</form>
