The light switch component is a block element. The label is enclosed within it so that it will always be on the same line as its corresponding switch. The default switch resembles a light switch, dark and gray when off and golden when on. Colors are customizable along with the default skins that come with the element.

Demo:
Light Switch
Off On
Markup:
<div class="switch">
  <div class="switch__title">Label</div>
  <input type="checkbox" aria-label="">
  <span>
    <span>Off</span>
    <span>On</span>
    <a></a>
  </span>
</div>

Inline

If you want a switch component to be inline with another element use .switch--inline class instead.

Demo:
Light Switch
Off On
Light Switch
Off On
Markup:
<div class="switch--inline">...</div>

Skins

All basic skins are available for this component.

Add the skin class to the input element inside the switch for the class to take affect.

Demo: switch--{{ color }}
Light Switch
Off On
Markup:
<div class="switch">
  <div class="switch__title">Label</div>
  <input type="checkbox" class="skin-{{ '{' }}{{ '{' }} color {{ '}' }}{{ '}' }}" aria-label="">
  ...
</div>

Color Customization

To customize a switch with a specific color that isn't part of our default skins I recommend creating a new skin class.

<div class="switch custom-skin">

Inside your stylesheet target these elements:

  • .switch input:checked + span : targets the background color of the switch when it's on
  • switch input:checked + span a : targets the color of the circle when the switch is on
