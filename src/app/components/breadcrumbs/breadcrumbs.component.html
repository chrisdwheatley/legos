Breadcrumb

Navigation is at the core of a good user experience in a website or web app. Breadcrumbs allow a user to visually see where they came from, how they got there and gives them a chance to easily go back a few steps if necessary. It can be infuriating if you’ve ever wanted to go back one or two screens but haven’t been given a menu option to do so and end up clicking Home and starting over again. Breadcrumbs help prevent those situations. They allow site visitors to always know where they are compared to where they've been.

Best practice: Use an ordered list for this component because it is a list and it does matter what order the list is in.

Demo:
  1. Home
  2. About Us
  3. Section 1
Markup:
<ol class="breadcrumb">
  <li class="breadcrumb__item">
    <a href="#">Home</a>
  </li>
  <li class="breadcrumb__item">
    <a href="#">Library</a>
  </li>
  <li class="breadcrumb__item active">
    Data
  </li>
</ol>

Skins

The basic skins are available for this component. Add .breadcrumb--{{ '{' }}color{{ '}' }} with .breadcrumb class. This will alter the color used for the active breadcrumb__item.

* NOTE: There is no gray color because the default color for previous breadcrumb links is already gray.

Demo:
  1. Home
  2. Exercises
  3. {{ color }}
Markup:
<ol class="breadcrumb breadcrumb--{{ color }}">...</ol>
