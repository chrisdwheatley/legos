breadcrumbs

Breadcrumbs are used to help with navigation within an application. Have you ever just wanted to back a screen or two but you can't figure out how so you just end up clicking home and starting over again. Breadcrumbs help prevent those situations. They allow site visitors to always know where they are in relation to where they've been.

Best practices: Use an ordered list for this component because it is a list and it does matter what order the list is in.

Demo:
  1. Home
  2. About Us
  3. Section 1
Markup:
<ol class="breadcrumb">
  <li class="breadcrumb__item">
    <a href="#">Home</a>
  </li>
  <li class="breadcrumb__item">
    <a href="#">Library</a>
  </li>
  <li class="breadcrumb__item active">
    Data
  </li>
</ol>

Skins

The basic color skins are availible for this component. Just add the .breadcrumb--{{ '{' }}color{{ '}' }} with the .breadcrumbclass. This will just alter the color used for the active breadcrumb__item.

* NOTE: There is no gray default color because the defaut color for the previous breadcrumb links are already gray.
