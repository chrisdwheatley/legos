Pagination

For showing multiple pages of related content. Hopefully these classes can help simplify the process.

Attributes:
  • role* : 'navigation'
  • aria-label : defines a label for the element (only used when label text not visible on screen)

* Role needs to be used for the nav as well as the list items inside. There are many differnt roles that you can use, see here for a list. For this component we will use the role="button" because all of our list items are buttons.

Accessibility

For accessibility purposes, the pagination component should be wrapped with the <nav> element to identify it as a navigation section. It is also possible and highly likely that a page may have more than one <nav> element already (such as the primary navigation in the header, or sidebar navigation). Best practice dictates providing a descriptive aria-label for the <nav> element signaling to screen readers and other assistive technologies what the element is.

Example: If the pagination component is used to navigate between a set of pages in a signup form, an appropriate label could be aria-label="sign up form pages".

Prev/Next Labels

There are customizations for how to represent the previous/first and next/last items. Either choose to have all of them display, or just choose one set. Either way, both symbols, and words are allowed. The only thing that changes when you add --first or --last to .pagination__item is the spacing. The first and last buttons have a margin that separates them from the inner set of buttons.

Demo:
First « 1 2 3 4 5 » Last
Markup:
<nav class="pagination" role="navigation">
  <span class="pagination__item--first" role="button">
    <a href="" aria-label="First">
      <span aria-hidden="true">First</span>
    </a>
  </span>
  <span class="pagination__item" role="button">
    <a href="" aria-label="[Previous or < symbol]">
      <span aria-hidden="true">«</span>
    </a>
  </span>
  ...
  <span class="pagination__item" role="button">
    <a href="" aria-label="[Next or > symbol]">
      <span aria-hidden="true">»</span>
    </a>
  </span>
  <span class="pagination__item--last" role="button">
    <a href="" aria-label="Last">
      <span aria-hidden="true">Last</span>
    </a>
  </span>
</nav>

Skins

All basic skins are available for this component.

Add the skin class to the main pagination element for the class to take affect.
