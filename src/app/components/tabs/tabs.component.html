tab

Usually, tabs are created with Javascript via a framework like Angular or React, taking advantage of ng-show, ng-hide, and ng-if but if you're looking for a pure CSS version it is possible. The default style of our tabs has no visual container around them. There are no borders, just a bottom border on the tab that is currently selected.

You can choose where the blue line is, either top or bottom by putting --top or --bottom on the label element.

Markup:
<div class="tab">
  <input id="tab-1" type="radio" name="tabs" checked>
  <label for="tab-1" class="border--bottom">
    <i class="icon fas fa-code"></i>
    <span>Tab Title</span>
  </label>
  ... more tabs here
  <section id="content-1" class="tabs__content">
    <h3>Headline 1</h3>
    <p>Content goes here...</p>
  </section>
  ... more sections here
</div>

Tab Amount

Depending on how many tabs are needed you might want to customize your CSS. By default three tabs are generated for use but if you need more than that add the below code to your Sass file to generate more.

$tabCount designates how many tabs you're going to need and how many should be generated in your CSS file.

$tabCount: 5;

.show--tab--content {{ '{' }}
  display: block;
  padding: 20px;
  background: #fff;
{{ '}' }}

@for $i from 1 through $tabCount {{ '{' }}
  #tab-#{{ '{' }}$i{{ '}' }}:checked ~ #content-#{{ '{' }}$i{{ '}' }} {{ '{' }}
    @extend .show--tab--content;
  {{ '}' }}
{{ '}' }}

// if you don't need the bordered version then you don't need this
.tab__container--bordered {{ '{' }}
  @for $i from 1 through $tabCount {{ '{' }}
    #tab-#{{ '{' }}$i{{ '}' }}:checked ~ #content-#{{ '{' }}$i{{ '}' }} {{ '{' }}
      @extend .show--tab--content; 
    {{ '}' }}
  {{ '}' }}
{{ '}' }}
