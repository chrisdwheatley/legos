Tab

Usually, tabs are created with Javascript via a framework like Angular or React. If you're looking for a pure CSS version and you don't need a complex large amount of tabs here's a solution. The default style has no visual container, just a bottom border on the tab that's currently selected.

The tab component uses the input and section elements id attribute to make the tabs dynamically show and hide the current tab content.

Customize where the active border is on the active tab, top or bottom, by putting --top or --bottom on the label element.

Markup:
<div class="tab">
  <input id="tab-1" type="radio" name="" aria-label="" checked>
  <label for="tab-1" class="border--bottom">
    <i class="icon fas fa-code" aria-hidden="true"></i>
    <span>Tab Title</span>
  </label>
  ... more tabs here
  <section id="content-1" class="tabs__content">
    <h3>Headline 1</h3>
    <p>Content goes here...</p>
  </section>
  ... more sections here
</div>

Skins

Tab skins have specific styles. They are very similar to global skins with additional characteristics.

Add the skin to the .chip element. The skin will also alter the close icon color if present and shadow on the chip.

Tab Amount

Depending on how many tabs are needed you might need to customize your CSS. By default three tabs are generated for use but if you need more than that add the below code to your Sass file to generate more.

$tabCount designates how many tabs you're going to need and how many should be generated in your CSS file.

$tabCount: 6;

.show--tab--content {{ '{' }}
  display: block;
  padding: 20px;
  background: #fff;
{{ '}' }}

// generates two new classes for extra tabs
@for $i from 1 through $tabCount {{ '{' }}
  #tab-#{{ '{' }}$i{{ '}' }}:checked ~ #content-#{{ '{' }}$i{{ '}' }} {{ '{' }}
    @extend .show--tab--content;
  {{ '}' }}
{{ '}' }}

// if you don't need the bordered version then you don't need this
.tab__container--bordered {{ '{' }}
  @for $i from 1 through $tabCount {{ '{' }}
    #tab-#{{ '{' }}$i{{ '}' }}:checked ~ #content-#{{ '{' }}$i{{ '}' }} {{ '{' }}
      @extend .show--tab--content; 
    {{ '}' }}
  {{ '}' }}
{{ '}' }}
