Chip

Chips are pill-shaped components that can be used in several ways. Some examples are list items, labels, or as a counter. They can remain static elements or allow users to interact and remove them.

To create chips you need to house them in a .chips container. This will keep them all together. Each chip gets the .chip class. Chips are bare by default leaving room for the skin to be applied.

Demo:
{{ chip[1] }}
Markup:
<div class="chips">
  <span class="chip">Hockey</span>
</div>

Removable (Optional)

You may want to allow the user to remove chips. Add the --removable modifier to the .chip class and add the icon element after your chip text. This will add the "x" icon to the end of the chip and allow the user to delete the chip.

Demo:
Hockey
Markup:
<div class="chips">
  <span class="chip--removable">
    Hockey
    <i class="icon fas fa-times chip__close-btn" aria-hidden="true" aria-label=""></i>
  </span>
</div>

Skins

Chips have their own specific skin styles. They are very similar to the global skins but with additional styles just for them. The skin class controls background color, font color, shadow color, border color, close button color (optional).

Add a skin to the .chip element. The skin will also alter the close icon color if present and shadow on the chip.

Demo:
chip--{{ chip[0] }} chip--outlined-{{ chip[0] }}
Markup:
<div class"chips">
  <span class="chip chip--green">Hockey</span>
  <span class="chip chip--outline-blue">Lacrosse</span>
<div/>
