Chips

Basic

Chips are pill shaped elements. They can be used for is several ways, such as list items, labels, or as counters for example. They can be static elements or you can allow users to interact and remove them.

To create chips you need to house them in a .chips container that keeps them all together in a wrapping container. Each individual chip gets the .chip class applied. Chips are bare by default leaving room for the skin to be applied onto of it.

Demo:
Hockey Lacrosse Baseball
Markup:
<div class="chips">
  <span class="chip">Hockey</span>
</div>

Skins

On the chip you add the skin color. This will alter the background-color, text color, border color, and if the chips are removable then the close icon color as well. To add a skin to the chip add it with the .chip class. There are two options for skins that can be applied to chips. Just like other elements, there is a filled skin and and outlined skin. See below for examples.

Demo:
No Skin Hockey Lacrosse Baseball Football
No Skin Hockey Lacrosse Baseball Football
Markup:
<div class"chips">
  <span class="chip skin-green">Hockey</span>
  <span class="chip skin-outline-blue">Lacrosse</span>
<span class="chip skin-red">Baseball</span>
<div/>

Removable Add-On

It is possible that you want the user to use the chips for a list and be able to remove items as they choose. To do that add the --removable modifier to the .chip class and add the icon element after your chip text. This will add the "x" icon to the end of the chip and allow the user to delete the chip.

Demo:
Hockey Lacrosse Baseball
Markup:
<div class="chips">
  <span class="chip--removable">
    Hockey
    <i class="icon fas fa-times chip__close-btn"></i>
  </span>
</div>
