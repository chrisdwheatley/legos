Card

The card component is built with content in mind. It had to be developed to be extremely versatile to accommodate as many different types of content in any way a user wants. With that in mind, the first demo is a classic representation of a card.

Demo:

Markup:
<div class="card">
  <div class="card__body">
    <span class="card__title">card title</span>
    <span class="card__text">card text</span>
  </div>
</div>

Next, we'll step through what options we can configure.

Images

Add an image to either the top or bottom of a card by adding an <img> or <div> element with the .card__display--top or .card__display--bottom class making sure to put it above or below the .card__body section.

Sizes

A card comes in three sizes. You can also customize the sizes on your own. The size of a card, if you don't add a specific size class defaults to medium. To add a size to a card add .card--sm or .card--lg.

Alignment

Basic alignment is built-in. Add .card--left, .card--centered, or .card--right on the card element for positioning. Anything further then you'll need to add more custom styles to individual items.
