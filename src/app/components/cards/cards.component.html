Card

Card components are built around a content-first approach. They are extremely versatile to accommodate as many different types of content in any order a user wants.

Demo:
300 x 168
Card Title This is content for a card component. The summary for the card goes here, which should be two to three lines maximum.
Markup:
<div class="card">
  <div class="card__body">
    <span class="card__title">card title</span>
    <span class="card__text">card text</span>
  </div>
</div>

Next, we'll step through card options and how we can configure them.

Images

Display an image or div element with the .card__display--top or .card__display--bottom classes. Make sure the image is inside the .card__body element.

Sizes

Card components have three sizes available. The size of a card, if you don't add a specific class the default width is 300px. Add .card--sm or .card--lg to the .card element to adjust.
