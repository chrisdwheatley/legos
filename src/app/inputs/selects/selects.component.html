native selects

Note that many native select menus—namely in Safari and Chrome—have rounded corners that cannot be modified via border-radius properties.

We have a custom block level select: .select

As wells as an inline select: .select-inline

Add any of the custom button colors to the .select wrapper.

<div class="select or select-inline">

  <select name="selectExample" id="selectExample">

    <option>Choose an option</option>

    <option value="1"></option>

    <option value="2"></option>

    <option value="3"></option>

  </select>

</div>

To add a color class or a button class you need to apply the classes in the following places. The .skin-{{color}} goes on the select wrapper and the .skin-{{size}} goes on the select itself. Example below.

<div class="select skin-green">

  <select name="selectExample" id="selectExample" class="skin-md">

    <option>Choose an option</option>

    <option value="1"></option>

    <option value="2"></option>

    <option value="3"></option>

  </select>

</div>
