Native Selects

Note that many native select menus—namely in Safari and Chrome—are not customizable. They have rounded corners that cannot be modified via border-radius properties etc. With our select menu we can customize whether you want rounded or squared corners. Our default is rounded but with the addition of class .select-squared you call change the border radius.

Block Level

Use this when you want the select to take up the entire container it's in or take up the entire line.

Demo:
Markup:
<div class="select">
<select name="name" id="id">
  <option>Choose an option</option>
  <option value="1"></option>
  <option value="2"></option>
  <option value="3"></option>
</select>
</div>
Inline

If you need to put a select menu next to another input or another button or input use the inline select.

Demo:
Markup:
<div class="select-inline">
<select name="name" id="id">
  ...
</select>
</div>
Squared Corners

For a customized square corner that is not the default add the .select-squaredclass to the select wrapper.

Demo:
Markup:
<div class="select select-squared">
<select name="name" id="id">
...
  </select>
</div>
Select Skins

Just like buttons, you can customize a select menu to have a skin color. To do this just add the skin class to the .select wrapper.

Demo:
Markup:
<div class="select skin-blue">
  <select name="name" id="ID">
    ...
  </select>
</div>
