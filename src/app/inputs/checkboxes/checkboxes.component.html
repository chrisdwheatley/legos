Native Checkbox

When we have checkbox inputs, by default, they are wrapped with a label element with a class of .checkbox. Be sure to add type="checkbox" as an attribute.

Demo:
Markup:
<label class="checkbox">
  <input type="checkbox" value="">Label 1
</label>

Native Inline Checkbox

Similar to our default checkbox inputs, modify the checkbox class and add --inline to the checkbox class on the label element for a series of checkboxes that you want to appear alongside eachother.

Demo:
Markup:
<label class="checkbox--inline">
  <input type="checkbox" id="" value="">Label Inline 1
</label>
<label class="checkbox--inline">
  <input type="checkbox" id="" value="">Label Inline 2
</label>
