Custom CSS Checkboxes

Need to play around with certain styles if you want to change the size.


<span class="custom-checkbox">

<input id="custom-checkbox1" type="checkbox" />

<label for="custom-checkbox1">

<span></span>

</label>

</span>

To change size and position of check inside the box change the top, left styles here:

input[type='checkbox']:checked + label > span:before {{'{'}}

top: 5px;

left: 2px;

{{'}'}}
