Input Groups

You can wrap input elements and buttons by wrapping them in .input--group. Use different btn classes for colors.

<div class="input--btn">

  <input type="text">

  <button type="button" class="btn btn--primary">Submit</button>

</div>



<div class="input--btn">

  <button type="button" class="btn btn--outline-default">Submit</button>

  <input type="text">

</div>

<div class="input--btn">

  <input type="text">

  <button type="button" class="btn btn--default">Submit</button>

</div>
