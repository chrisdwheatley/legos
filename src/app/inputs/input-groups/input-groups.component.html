Input Groups

You can combine input elements with buttons to get a input group. Wrap the input element and button with a .input-group. You still have the option to add on a btn color.

Demo:
Markup:
<div class="input-group">
  <input type="text" class="input-group__input">
  <button type="button" class="btn input-group__btn btn--primary">Submit</button>
</div>
