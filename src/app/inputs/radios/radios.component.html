Radio Groups

In our forms radio inputs are stacked by default. Just add a .radio class around a parent container for the inputs.

<div class="radio">

  <label>

    <input type="radio" name="" id="" value="" checked>Radio option 1

  </label>

</div>
Inline Radio Groups

Similarly use the .radio--inline class on a series of radio inputs that you want to appear alongside eachother.



<label class="radio--inline">

  <input type="radio" name="" id="" value=""> 1

</label>

<label class="radio--inline">

  <input type="radio" name="" id="" value=""> 2

</label>
