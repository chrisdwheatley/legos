Button Icon

Incorporate Font Awesome icons into your buttons or use an icon as a button. If you have a button that has an icon and text wrap the text in a <span> element to allow for sufficient padding between the icon and the text.

When using an icon as the only content in a button add an aria-label to assist screen readers in understanding the purpose of the button.

Attributes:
  • aria-label : defines a label for the element (only used when label text not visible on screen)
Demo:
Markup:
<button type="button" class="btn btn--primary">
  <i class="icon fas fa-cog" aria-hidden="true"></i>
  <span>Settings</span>
</button>
<button type="button" class="btn btn--primary" aria-label="" title="">
  <i class="icon fas fa-cog" aria-hidden="true"></i>
</button>

Upload Button

The upload button is special. We combine an <input> and <label> element. You can also add an icon inside. The .btn-upload class has the styles that make it look like a button. You can add the normal button color to it. Note where the button color class needs to go in the markup.

Demo:
Markup:
<div type="button" class="btn-upload-container">
  <input type="file" name="uploadFile" id="upload" class="inputfile" 
    tabindex="0" 
    accept=".xlsx" 
    required="required" 
    aria-required="true" 
    aria-invalid="true">
  <label for="upload" class="btn-upload btn--primary">
    <i class="icon fas fa-upload" aria-hidden="true"></i>
    <span>Choose a file</span>
  </label>
</div>
