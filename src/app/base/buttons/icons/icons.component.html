button with icon

Incorporate Font Awesome icons into your buttons or use an icon as a button. If you have a combination button that has an icon and a word wrap your text in a <span> to allow for sufficient padding between the icon and the word.

Demo:
Markup:
<button type="button" class="btn btn--primary">
  <i class="icon fas fa-cog"></i>
  <span>Settings</span>
</button>
<button type="button" class="btn btn--primary" aria-label="settings button">
  <i class="icon fas fa-cog"></i>
</button>

Upload Button

The upload button is a special. We combine an <input> and <label> element. You can also add an icon inside. The .btn-upload class has the styles that make it look like a button. You can add the normal button color to it. Note where the button color class needs to go in the markup.

Demo:
Markup:
<div type="button" class="btn-upload-container">
  <input type="file" name="uploadFile" id="upload" class="inputfile" 
    tabindex="0" 
    accept=".xlsx" 
    required="required" 
    aria-required="true" 
    aria-invalid="true">
  <label for="upload" class="btn-upload btn--primary">
    <i class="icon fas fa-upload"></i>
    <span>Choose a file</span>
  </label>
</div>
