Dropdowns

Our dropdowns are solely built with CSS without any need for Javascript. Under the hood, they are activated with an invisible input checkbox. Because of this if the dropdown is open you need to click it again to close it. Clicking outside of it will not close it, but you could add some custom javascript if that's your thing.
