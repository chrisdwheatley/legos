containers

Three classes can quickly address the layout of content. These are intended to be used for simple base layouts. For more complex containers you may need to add more specific CSS of your own.

Base Container

The .container class creates a responsive box and adds space to the bottom of each container.

* Note: If you nest a .container inside of another 25px of padding will be added to the left side of the container. This is to be used as indenting subsections of text or dividing parts of a form, as an example.

Markup:
position: relative
margin: 0 auto 20px auto
padding: 0
width: 100%

Flex Row

Use the .flex-container class as it's own flex container (default row), alongside the base container class to turn that container into a flex parent. This flex container uses the styles below.

Markup:
display: flex
flex-wrap: wrap
justify-content: flex-start
align-items: center

Flex Column

Use the .flex-column class as it's own flex container with a column direction. It does not share styles with the .flex-container class. Like the .flex-container this can be used with or without the base .container class. This flex container uses the styles below.

Markup:
display: flex
flex-direction: column
