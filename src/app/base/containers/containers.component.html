Container

Three classes can quickly address the layout of your content. These are intended to be used for simple base layouts. For more complex containers you may need to add more specific CSS of your own.

Base Container

The .container class creates a responsive box with space on the bottom. The max-width of these containers is 600px. That width is a good length for paragraph width to keep text readable.

* Note: If you nest a .container inside of another, 25px of padding will be added to the left side of the inner container. This is intended to be used for indenting subsections of text or dividing parts of a form.

Markup:
position: relative
margin: 0 auto 20px auto
padding: 0
width: 100%
max-width: 600px

Flex Row

Use the .flex-container class as it's own flex container (default row), alongside the base container class to turn that container into a flex parent. This flex container uses the styles below.

Markup:
display: flex
flex-wrap: wrap
align-items: center

Flex Column

Use the .flex-column class as it's own flex container with a column direction. It does not share styles with the .flex-container class. Like the .flex-container this can be used with or without the base .container class. This flex container uses the styles below.

Markup:
display: flex
flex-direction: column
