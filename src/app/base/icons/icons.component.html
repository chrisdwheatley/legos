Icon

LegoCSS uses Font Awesome icons. Visit the website to see all icons that are available. You can use the icons anywhere using a span or i elements. By default, the icons have a height of 14px.

See this article for more information on Font Awesome icon accessibility.

Attributes:
  • aria-hidden : 'true'
  • title : it is best practice to add a title which provides a tooltip for sighted mouse users
Demo:
Markup:
<i class="icon fas fa-shower"></i>

Decorative:

If the icon you are using is purely decorative, has no functional purpose, and will not effect a users interacts then the icon is decorative. In this case add an aria-hidden attribute to the icons so screen readers don't waste time trying to read what it is.

Demo:
Markup:
<i class="icon fas fa-shower" aria-hidden="true"></i>

Semantic:

If an icon is being used to convey meaning, such as an button icon with no text associated with it, it is a semantic icon. There are a few things you need to do to make sure this icon is accessible for assisitive technologies.

If the icon is wrapped by another element such as a button or an anchor element then adding the aria-label to the outer element is suffiecient.

Attributes:
  • aria-label : adding a description to the element will provide an accessible alternative name for the element
Demo:
Markup:
<button ... aria-label="" title="">
  <i ... aria-hidden="true"></i>
</button>

OR

<i ... aria-hidden="true" aria-label="" title=""></i>
