Icon

LegoCSS uses Font Awesome for icons. Visit the website to see all available icons.

You can use the icons anywhere using the <span> tag. By default, the icons have a height of 14px. If you would like to customize the sizing there are additional class options that you can add on.

You can customize the sizing of your icon with the additional size classes below.

Demo:

Markup:
<i class="icon fas fa-shower" aria-hidden="true"></i>
<i class="icon fas fa-shower fa-lg" aria-hidden="true"></i>
<i class="icon fas fa-shower fa-2x" aria-hidden="true"></i>
<i class="icon fas fa-shower fa-3x" aria-hidden="true"></i>
<i class="icon fas fa-shower fa-4x" aria-hidden="true"></i>
<i class="icon fas fa-shower fa-5x" aria-hidden="true"></i>
