lists

There are a few different ways to style a list. The default list is a bulleted with only slight padding on the left.

For more information about accessability and WAI-ARIA have a look at what the W3C says about it, Accessible Rich Internet Applications (WAI-ARIA) 1.2.

ARIA Roles:

directory, group, listbox, menu, menubar, radiogroup, tablist, toolbar, tree, presentation

Unstyled

For when you want your list without a style but still want some spacial seperation from the left side.

Demo:
  • unstyled list item 1
  • unstyled list item 2
  • unstyled list item 3
Markup:
<ul class="list--unstyled" role="group">
  <li role="groupitem">unstyled list item 1</li>
  <li role="groupitem">unstyled list item 2</li>
  <li role="groupitem">unstyled list item 3</li>
</ul>

Numbered

For when you need a list when the order matters and you want your user to know it matters.

Demo:
  1. numbered list item 1
  2. numbered list item 2
  3. numbered list item 3
Markup:
<ol class="list--numbered" role="group">
  <li role="groupitem">numbered list item 1</li>
  <li role="groupitem">numbered list item 2</li>
  <li role="groupitem">numbered list item 3</li>
</ol>

Part of navigation

This type of list is used for most navigations. As you can see also indicated by the role attribute.

Demo:
  • nav list item 1
  • nav list item 2
  • nav list item 3
Markup:
<ul class="list--nav" role="menu">
  <li role="menuitem">nav list item 1</li>
  <li role="menuitem">nav list item 2</li>
  <li role="menuitem">nav list item 3</li>
</ul>
