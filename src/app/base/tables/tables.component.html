table

Basic

The basic table component is used for tabular data, not layout. For basic styling use <table> tag. There are no column dividers on the default table, only row.

Best Practices: Use the tag instead not a class name. For accessibility if you are going to be displaying tabular data then use the <table>, <thead>, <tbody>, <tr>, <td>, and <caption> where appropraite. Screen readers and assistive technologies cannot read class names and decided what to do with it.

Demo:
This is an example of a table caption
# Header Column 1 Header Column 2 Header Column 3
1 Body Column 1 Body Column 2 Body Column 3
2 Body Column 1 Body Column 2 Body Column 3
3 Body Column 1 Body Column 2 Body Column 3
Markup:
<table>
  <caption>table caption</caption>
  <thead>
    <tr>
      <td>#</td>
      <td>Header Column 1</td>
      <td>Header Column 2</td>
      <td>Header Column 3</td>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>1</td>
      <td>Body Column 1</td>
      <td>Body Column 2</td>
      <td>Body Column 3</td>
    </tr>
    ...
  </tbody>
</table>
