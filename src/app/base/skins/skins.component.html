Skin

Skins are global classes that can be added to many of the components in the LegoCSS library. Visit the component page to see how to implement each skin.

Colors

  • skin-{{ color }}

Skins control:

  • Backgroud Color
  • Font Color
  • Border Color (optional add-on)

Components

  • Select Input
  • Accordion
  • Breadcrumb
  • Chips
  • Switch

Not Availible For:

Some components can not use the global skins as they need more specific styles for their skins. The below components have their own skin classes specific to their components. See their pages for more information.

  • Button
  • Alert
  • Pagination
