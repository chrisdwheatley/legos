inputs

This is what the most used inputs look like by default. By changing the type attribute these <input> elements take on different roles.

For a comprehensive list of attributes see the MDN Web Docs.

Attributes:
  • type : the most important and most powerful attribute for an input element
  • id : match this attribute with the corresponding for attribute that is on the <label> element
  • title
  • placeholder
