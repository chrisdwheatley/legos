radio

Radio elements, by default, are wrapped with a label element and a .radio class. Be sure to add type="radio" as an attribute.

Demo:
Markup:
<label class="radio">
  <input type="radio" name="" id="" value="" checked>Label 1
</label>

Radio Inline

To modify the radio elements to be inline with eachother add --inline to the radio class on the label element.

Demo:
Markup:
<label class="radio--inline">
  <input type="radio" name="" id="" value="">Label 1
</label>
<label class="radio--inline">
  <input type="radio" name="" id="" value="">Label 2
</label>
