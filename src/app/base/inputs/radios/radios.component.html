Radio

Radio elements, by default, are wrapped with a label element and a .radio class. Be sure to add type="radio" as an attribute.

A basic radio has an input wrapped by a label.

Demo:
Markup:
<label class="radio" for="">
  <input type="radio" name="" id="" checked>Label 1
</label>

Radio Group Inline

It is best practice to enclose radio inputs inside of a input-group element. Wrapping the radios keeps the corresponding question inline with the radio elements.

Demo:
Markup:
<div class="input-group">
  <label aria-label="">The question for radio:</label>
  <label class="radio" for="">
    <input type="radio" name="" id="" checked>Radio 1
  </label>
  <label class="radio" for="">
    <input type="radio" name="" id="" checked>Radio 2
  </label>
</div>

Radio Group Stacked

If you'd like to use this in a form or with a label for a question it is wrap each section in a input-group and put the question label inside. The radio inputs are already wrapped by their label.

Markup:
<div class="input-group--stacked">
  <label class="radio" for="">
    <input type="radio" name="" id="" checked>Radio 1
  </label>
  <label class="radio" for="">
    <input type="radio" name="" id="" checked>Radio 2
  </label>
</div>
