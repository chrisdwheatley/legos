Native Radio

When we have radio inputs, by default, they are wrapped with a label element with a class of .radio. Be sure to add type="radio" as an attribute.

Demo:
Markup:
<label class="radio">
  <input type="radio" name="" id="" value="" checked>Label 1
</label>

Native Inline Radio

Similar to our default radio inputs, modify the radio class and add --inline to the radio class on the label element for a series of radio that you want to appear alongside eachother.

Demo:
Markup:
<label class="radio--inline">
  <input type="radio" name="" id="" value="">Label 1
</label>
<label class="radio--inline">
  <input type="radio" name="" id="" value="">Label 2
</label>
