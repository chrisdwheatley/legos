Checkbox

Checkbox elements, by default, are wrapped with a label element and a .checkbox class. Be sure to add type="checkbox" as an attribute.

Demo:
Markup:
<label class="checkbox">
  <input type="checkbox" value="">Label 1
</label>

Checkbox Inline

For a series of checkbox inputs all on the same line modify the checkbox class adding --inline.

Demo:
Markup:
<label class="checkbox--inline">
  <input type="checkbox" id="" value="">Label Inline 1
</label>
<label class="checkbox--inline">
  <input type="checkbox" id="" value="">Label Inline 2
</label>
