select

Note that many native select menus—namely in Safari and Chrome—are not customizable. They have rounded corners that cannot be modified via the border-radius property. With our custom select element, we can customize whether you want rounded or squared corners.

Attributes:
  • name : select element, specifies the name of the control group
  • multiple : select element, indicates whether more than option is allowed to be selected
  • label : option element, if there is not label present then the value becomes the label text
  • value : option element, the value to be submitted with the form
  • aria-label : defines a label for the element (only used when label text not visible on screen)

Block Level

When you want the select element to take up its entire parent container or span the entire line.

Demo:
Markup:
<div class="select">
  <select name="name" id="id">
    <option>Choose an option</option>
    <option value="1"></option>
    <option value="2"></option>
    <option value="3"></option>
  </select>
</div>

Inline

If you want to put a <select> element next to another <input> element or <button> use the .select--inline class.

Demo:
Markup:
<div class="select--inline">
  <select name="name" id="id">
    ...
  </select>
</div>

Squared Corners

To change the corners of the <select> element to be squared add the .select--squared class to the <select> wrapper.

Demo:
Markup:
<div class="select select--squared">
  <select name="name" id="id">
    ...
  </select>
</div>

Select Skins

Just like buttons, you can customize a select to have a skin color. To do this add the skin class to the .select wrapper.

Demo:
Markup:
<div class="select skin-blue">
  <select name="name" id="ID">
    ...
  </select>
</div>
