documentation

The LegoCSS documentation is sectioned out by component. Each component has the below layout.

  • description
  • demo
  • markup

In each section, there is a small explanation of the component, how it's made, a demo of the element, and some example code showing how to use it. When there are customizations like skins available those will also be shown.

The demos allow you to play with each component, look under the hood with the developer tools and get to see a working example. Along with the developer tools, example markup is also provided with highlighted classes or elements that are important to each component.

Be sure to notice when I highlight parts of the code. I do this to show you specific classes that pertain to that component.
