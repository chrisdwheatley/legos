Documentation

This documentation is sectioned out by component.

Documentation structure:

  • Description
  • Demo
  • Code

Each section contains a small explanation of how it's made, a demonstration of the element to play with, and an example code piece showing how it is written. When there are customizations like skins available those will also be shown.

The demo allows you to play with each component, look under the hood with developer tools and see a working example. Accompanying each demo are code examples highlighting classes or elements that are important to each component.
