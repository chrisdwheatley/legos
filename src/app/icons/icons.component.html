This library uses Font Awesome. If you want to checkout all the icons you have avalible to you go here: Font Awesome.

You can use the icons just about anywhere with the <i> tag or a <span>. By deafault the icons have a height of 14px. If you would like to customize the sizing there are some aditional class options availible to you.

For convenience most icons are usually next to some text. We have some classes that you can add onto you icon to give it one directional padding, or padding on both sides. Use .icon--pad--left for padding-left: 5px; or .icon--pad--right for padding-right: 5px;. Or if you want padding on both sides of the icon use .icon--pad for padding: 0 5px;. This can be helpful when you have an icon inside a button next to some text.
